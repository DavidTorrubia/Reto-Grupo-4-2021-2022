------------------------ Set up guide ------------------------
Este proyecto esta diseñado en php usando el gestor Xampp
Para poner en marcha el proyecto: 

	1- instale la ultima version de xampp
	2- Localice la carpeta htdocs en la carpeta de instalacion de xampp
	4- Acceda a xampp e inicie "Apache" y "Mysql"
	5- En un buscador acceda a "http://localhost/phpmyadmin/"
	6- Cree una base de datos con nombre "reto" sin las comillas y en minuscula
	7- En la pestaña SQL introduzca el contenido del archivo de "creacionDBTotal.txt" 		localizado en la carpeta "Base de datos" dentro de documentacion
	8- Una vez hecho todo esto podra acceder al index introduciendo este enlace en el 		buscador "http://localhost/Reto/Content/HTML/LandingPage.php"

------------------------ Bases de datos ------------------------	
Se dispone de un archivo excel que contiene toda la informacion de las cuentas de prueba introducidas en la base de datos en la carpeta "Base de datos" llamado "UsuariosYContraseñas.xlsx"
